# 모두의공영주차장 — 로컬 실행

## 실행 (Python 3만 있으면 됨)

```bash
cd 모두의공영주차장
python3 server.py
```

브라우저에서 **http://localhost:8765** 접속.

## 실제 데이터 전체 받기 (필수 1단계)

지금은 `config.json`의 키가 `"sample"`이라 **API별 5건만** 나옵니다(서울시 정책).
전체 데이터(실시간 123곳 + 전체 2,204곳)를 보려면:

1. https://data.seoul.go.kr 회원가입 → 로그인
2. 마이페이지 → **인증키 신청** (무료, 즉시 발급)
3. `config.json`의 `"seoulKey": "sample"` 을 발급받은 키로 교체
4. 서버 재시작

## 구조

| 파일 | 역할 |
|---|---|
| `server.py` | 로컬 서버: 정적 서빙 + `/api/parking` 프록시(CORS 우회, 5분 캐시) |
| `index.html` | 앱 화면 (F0~F4: 현위치·혼잡도 지도·실시간·거주자우선·상세) |
| `config.json` | 서울시 인증키·포트 |
| `coords_override.json` | 실시간 API에 좌표가 없어 매칭 실패 시 쓰는 보정 좌표 |

## 데이터 소스

- 실시간 주차대수: 서울 열린데이터광장 `GetParkingInfo` (시영주차장 ~123곳, 5분 캐시)
- 전체 주차장 정적정보: `GetParkInfo` (~2,204곳, 요금·운영시간·좌표)
- 두 API를 이름 정규화로 매칭해 통합. 좌표 없는 곳은 리스트에만 표시

## 아직 안 되는 것 (다음 단계)

- **거주자우선 표시**: 전국거주자우선주차정보표준데이터 필요 → data.go.kr에서 활용신청 후 연동 예정 (현재 키 미등록 상태)
